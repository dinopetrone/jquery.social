jQuery Social Jquery plugin for handling those annoying social networks you have to add everywhere

Facebook

Faceook is pretty basic as the sharing url only accepts one param and then reads page meta for the rest of it's result. Ensure your meta is setup correctly to get this working well. View this page source for an example.

ex:

$('.fb').facebook()

Options + Defaults

  • shareurl: location.href
  • windowName: 'facebook-share-dialog'
  • width:626
  • height:245

Twitter

Twitter is also fairly basic, you really only NEED to send one param of message

ex:

$('.tw').twitter({message:"Tweet goes here it can inclue #hash's as well"})

Options + Defaults

  • message: 'options.message Tweet Text'
  • windowName: 'twitter-share-dialog'
  • width: 626
  • height: 260

Pinterest

Pinterest is a bit more complex because you are basically pinning an image, i wanted to keep it simple so this only requires description and will grab the facebook og:image if you don't pass an image as an option

ex:

$('.pt').pinterest({description:'Pin goes here'})

Options + Defaults

  • description:'options.description Pinterest Description'
  • windowName: 'pinterest-share-dialog'
  • width:626
  • height:260
  • href:location.href
  • origin:location.origin
  • shareImg:$("meta[property='og:image']").attr("content")
